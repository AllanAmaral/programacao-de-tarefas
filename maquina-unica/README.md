# Máquina Única (1)

Esta pasta reúne implementações para o problema de programação de tarefas no ambiente de máquina única.

As notações e a nomenclatura utilizadas nas pastas `modelos-exatos`, `heuristicas` e `gerador-de-instancias` estão de acordo com a literatura utilizada em seu desenvolvimento. Para aprofundamento nos fundamentos teóricos do problema, recomenda‑se a consulta às obras referenciadas em cada arquivo.
