# Programação de Tarefas (Scheduling)

Esta pasta contém implementações para a solução do problema de programação de tarefas em diferentes ambientes de máquinas, organizadas da seguinte forma:

- Máquina única
- Máquinas paralelas
- Flow shop
- Job shop

Dentro de cada ambiente, os arquivos estão separados em:

- Modelos exatos
- Heurísticas
- Instâncias
